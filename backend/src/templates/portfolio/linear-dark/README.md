# Linear Dark

A polished dark portfolio template designed for product teams, makers, and SaaS builders. It combines a muted dark canvas with rich mesh gradients, glassmorphism panels, and subtle keyboard shortcut hints to evoke a premium product interface without copying any existing brand.

## Features

- **Dark product UI** with deep background and soft accent glow
- **Glassy cards** for projects, skills, and timeline entries
- **Keyboard shortcut badges** rendered in CSS-only layout
- **Status indicators** for shipped, building, and research states
- **Responsive layout** optimized for desktop, tablet, and mobile
- **Accessible interactions** with visible focus states and smooth scroll behavior
- **Minimal JS** for placeholder injection and mobile nav toggling

## Usage

This theme uses a small `PORTFOLIO` object in `script.js` to inject user data into the HTML placeholders. Update the object values to personalize the portfolio content.

### Files

- `index.html` — page structure and placeholders
- `style.css` — theme styling, layout, responsive breakpoints, and visual system
- `script.js` — data injection, skills/projects/experience rendering, and interaction behavior
- `meta.json` — metadata used by the portfolio template loader

## Customize

- Replace placeholder text in `script.js`
- Update accent colors and spacing in `style.css`
- Add or remove projects, skills, and experience items in `script.js`
